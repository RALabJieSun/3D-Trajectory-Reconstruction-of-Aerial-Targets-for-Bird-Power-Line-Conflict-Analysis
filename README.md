# 3D Trajectory Reconstruction of Aerial Targets in Transmission Line Scenarios

This repository is mainly used to present demo videos of aerial target tracking, binocular ranging, and 3D trajectory reconstruction in transmission line scenarios.

The work is developed for bird–power line conflict analysis. It focuses on reconstructing the 3D trajectories of aerial targets under complex outdoor backgrounds, so that target motion, line-crossing behavior, and spatial relationships with transmission lines can be observed more intuitively.

## What this repository shows

This repository is intended for **video demonstration** rather than full project reproduction.

The videos mainly show:

- aerial target detection and tracking in transmission line scenarios
- binocular local matching under detection-box constraints
- target ranging and 3D position estimation
- continuous 3D trajectory reconstruction in video sequences
- visualization results under complex outdoor backgrounds
